# Alexa plays Global Thermonuclear War

A simple Alexa skill to play "Global Thermonuclear War".

## Install

```sh
npm install
```

## Test

```sh
npm run test
```

## Build

```sh
npm run build
```

## Deploy

Install the [AWS CLI](http://docs.aws.amazon.com/cli/latest/userguide/installing.html) or `brew install awscli`.

Configure AWS CLI:

```sh
aws configure
```
